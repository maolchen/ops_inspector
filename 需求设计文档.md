# 运维巡检平台 - 需求设计文档

## 一、项目概述

### 1.1 项目背景
构建一个轻量级运维巡检平台，用于自动化收集 Prometheus 监控数据并生成可视化的巡检报告，支持多项目管理和灵活的巡检规则配置。

### 1.2 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                      前端层 (Vue 3)                      │
│  ┌─────────────┬─────────────┬─────────────┬──────────┐ │
│  │  项目管理   │  规则配置   │  巡检展示   │ 报告导出 │ │
│  └─────────────┴─────────────┴─────────────┴──────────┘ │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                    后端层 (Go + Gin)                     │
│  ┌─────────────┬─────────────┬─────────────┬──────────┐ │
│  │  项目模块   │  规则模块   │  巡检模块   │ 导出模块 │ │
│  └─────────────┴─────────────┴─────────────┴──────────┘ │
│  ┌────────────────────────────────────────────────────┐ │
│  │            Prometheus Client (数据采集)            │ │
│  └────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
                           │

│  │          │ groups   │          │ reports         │   │
│  └──────────┴──────────┴──────────┴─────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

### 1.3 技术选型

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|

#### 2.3.2 数据模型

```go
// Rule 巡检规则表
type Rule struct {
    ID             uint      `gorm:"primaryKey" json:"id"`
    GroupID        uint      `gorm:"index;not null" json:"group_id"`
    Name           string    `gorm:"size:100;not null" json:"name"`
    Type           bool      `gorm:"default:true" json:"type"` // true=告警规则, false=仅展示
    ShowInTable    bool      `gorm:"default:false" json:"show_in_table"`
    Description    string    `gorm:"size:500" json:"description"`
    Query          string    `gorm:"type:text;not null" json:"query"` // 即时查询 PromQL
    TrendQuery     string    `gorm:"type:text" json:"trend_query"` // 趋势查询 PromQL
    Threshold      *float64  `json:"threshold"` // 阈值，可为空
    Unit           string    `gorm:"size:20" json:"unit"` // 单位
    Labels         string    `gorm:"type:text" json:"labels"` // JSON 格式的标签别名映射
    ThresholdType  string    `gorm:"size:20;default:'greater'" json:"threshold_type"`
    ProjectScope   string    `gorm:"size:100;default:'*'" json:"project_scope"` // 适用项目，* 表示全部
    Enabled        bool      `gorm:"default:true" json:"enabled"`
    SortOrder      int       `gorm:"default:0" json:"sort_order"`
    CreatedAt      time.Time `json:"created_at"`
    UpdatedAt      time.Time `json:"updated_at"`
}

// ThresholdType 枚举值
const (
    ThresholdGreater      = "greater"        // 值 > 阈值 = critical
    ThresholdGreaterEqual = "greater_equal"  // 值 >= 阈值 = critical
    ThresholdLess         = "less"           // 值 < 阈值 = normal
    ThresholdLessEqual    = "less_equal"     // 值 <= 阈值 = normal
    ThresholdEqual        = "equal"          // 值 == 阈值 = normal
    ThresholdAtLeast      = "at_least"       // 值 >= 阈值 = normal
)
```

### 2.5 报告导出模块

#### 2.5.1 功能列表
- 导出为 PDF
- 导出为图片（PNG）
- 导出为 Word（可选）

#### 2.5.2 实现方案

**方案一：前端导出（推荐）**
- 使用 html2canvas 将页面转为图片
- 使用 jspdf 将图片生成 PDF
- 优点：实现简单，所见即所得
- 缺点：依赖浏览器渲染

**方案二：后端导出**
- 使用 Go 的 chromedp 库生成 PDF
- 或使用模板引擎生成 HTML 再转 PDF
- 优点：不依赖前端，更稳定
- 缺点：实现复杂度较高

**建议采用方案一**，前端实现更简单且效果好。

### 4.2 阈值判断逻辑

```go
func EvaluateStatus(value float64, threshold *float64, thresholdType string, isAlert bool) string {
    // 如果不是告警规则，直接返回正常
    if !isAlert || threshold == nil {
        return "normal"
    }
    
    criticalThreshold := *threshold
    warningThreshold := criticalThreshold * 0.9
    
    switch thresholdType {
    case "greater":
        // 值 > 阈值 = critical
        if value > criticalThreshold {
            return "critical"
        } else if value > warningThreshold {
            return "warning"
        }
        
    case "greater_equal":
        // 值 >= 阈值 = critical
        if value >= criticalThreshold {
            return "critical"
        } else if value >= warningThreshold {
            return "warning"
        }
        
    case "less":
        // 值 < 阈值 = normal
        if value < criticalThreshold {
            return "normal"
        } else if value < criticalThreshold * 1.1 {
            return "warning"
        }
        return "critical"
        
    case "less_equal":
        // 值 <= 阈值 = normal
        if value <= criticalThreshold {
            return "normal"
        } else if value <= criticalThreshold * 1.1 {
            return "warning"
        }
        return "critical"
        
    case "equal":
        // 值 == 阈值 = normal
        if value == criticalThreshold {
            return "normal"
        }
        return "warning"
        
    case "at_least":
        // 值 >= 阈值 = normal
        if value >= criticalThreshold {
            return "normal"
        } else if value >= warningThreshold {
            return "warning"
        }
        return "critical"
    }
    
    return "normal"
}
```

// 即时查询
func (c *PrometheusClient) Query(ctx context.Context, query string) (model.Value, error) {
    result, _, err := c.api.Query(ctx, query, time.Now())
    return result, err
}

// 趋势查询（最近7天）
func (c *PrometheusClient) QueryRange(ctx context.Context, query string) (model.Value, error) {
    end := time.Now()
    start := end.Add(-7 * 24 * time.Hour)
    step := time.Hour
    
    result, _, err := c.api.QueryRange(ctx, query, v1.Range{
        Start: start,
        End:   end,
        Step:  step,
    })
    return result, err
}

### 7.1 Prometheus 客户端集成

```go
package prometheus

import (
    "context"
    "time"
    
    "github.com/prometheus/client_golang/api"
    v1 "github.com/prometheus/client_golang/api/prometheus/v1"
    "github.com/prometheus/common/model"
)

type Client struct {
    api v1.API
}

func NewClient(address string, token string) (*Client, error) {
    cfg := api.Config{
        Address: address,
    }
    
    if token != "" {
        cfg.RoundTripper = &authTransport{
            token: token,
        }
    }
    
    client, err := api.NewClient(cfg)
    if err != nil {
        return nil, err
    }
    
    return &Client{
        api: v1.NewAPI(client),
    }, nil
}

// authTransport 添加认证头的 Transport
type authTransport struct {
    token string
}

func (t *authTransport) RoundTrip(req *http.Request) (*http.Response, error) {
    req.Header.Set("Authorization", "Bearer "+t.token)
    return http.DefaultTransport.RoundTrip(req)
}
```

async function exportToPDF(elementId) {

    scale: 2,
    useCORS: true
