## 关键说明

### 1. 基础资源详情
- 非磁盘列（IP、CPU、内存、TCP等）按节点**合并单元格**
- 磁盘列（挂载点、磁盘总量、磁盘使用量、磁盘使用率）**不合并**，每个挂载点一行
- 空白单元格表示"同上"（被合并）

### 2. K8S节点就绪状态
- 状态类型固定为 "Ready"
- 值=0 → 正常(绿色背景)
- 值≠0 → 异常(红色背景)

### 3. Pod运行状态
- 运行状态列显示查询出来的值（0或1）
- 值=1 → 正常(绿色)
- 值≠1 → 异常(红色)

### 4. PVC使用率
- 列顺序：PVC名称、命名空间、PVC使用率、状态
- 值>=90% → 异常(红色)
- 值<90% → 正常(绿色)

### 5. 证书状态
- 按证书类型分成独立表格
- 每个表格：节点、值、状态
- 值>=30天 → 正常，值<30天 → 异常

### 6. 进程表格
- 去掉"指标名称"列
- 列：进程名、所在机器、值、状态

### 7. 数据来源映射

| 表格 | 字段 | 数据来源 |
|------|------|----------|
| 节点就绪状态 | 节点 | labels.node |
| 节点就绪状态 | 值 | 查询结果值 |
| Pod运行状态 | 命名空间 | labels.namespace |
| Pod运行状态 | Pod名 | labels.pod |
| Pod运行状态 | 运行状态 | 查询结果值 |
| PVC使用率 | PVC名称 | labels.persistentvolumeclaim |
| PVC使用率 | 命名空间 | labels.namespace |
| PVC使用率 | PVC使用率 | 查询结果值 |
| 证书状态 | 节点 | labels.instance 或 labels.node |
| 证书状态 | 值 | 查询结果值（天） |
| 进程Top5 | 进程名 | labels.groupname |
| 进程Top5 | 所在机器 | labels.instance（去端口） |
| 进程Top5 | 值 | 查询结果值 |
